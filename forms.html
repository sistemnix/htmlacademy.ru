Связываем подпись и поле по id

Иногда обернуть поле и текст подписи в тег <label> нельзя.
Например, когда они размещены в разных ячейках таблицы
В этом случае можно связать подписьс полес с помощью атрибута <id>

    Алгоритм такой:
  1. Добавляем к полю ввода идентификатор с помощью атрибута id.
  2. Оборачиваем текст подписи в тег
  3. Добавляем тегу
  4. В атрибут for записываем такое же значение, что и в атрибуте id у поля.

Например:

<label for="user-field-id">Имя пользователя</label>
...
много-много других тегов
...
<input id="user-field-id" type="text" name="username">

Многострочное поле ввода

    Атрибуты тега <textarea>
  • id
  • name
  • rows(целое число) задаёт высоту многострочного поля в строках.
  • cols(целое число) задаёт ширину поля в символах. Ширина это некоторая
    "усреднённая величина"
  • !!!Значение "value" отсутствует, а знчение пол умолчанию задаётся текстом,
    расположенным между тегами <textarea>

Радиобаттон

Обычно переключатели размещают группами по несколько штук. Причём у переключателей
из олдной группы должно быть одинаковое имя и разные знаения, которые задаются с помощью "value".
Value является обязательным для переключателей.


Раскрывающийся список или «селект»

    <select>

      <option value="1">Вариант1</option>

      <option value="1">Вариант2</option>

    </select>

Позволяет выбрать один вариант из нескольких

    Атрибуты:
  1. name
  2. id

!!Если при отправке формы у выбранного варианта задан "value",
то на сервер отправится значение этого атрибута. В проитвном случае
будет отправлен текст подписи.!!

Мультиселект

<select multiple size="2"> -size - размер высоты селекта;

Любимые дни недели:

В мультиселекте можно выбрать несколько вариантов,
щёлкая по ним с зажатой клавишей Ctrl на Windows или Command на MacOS.
